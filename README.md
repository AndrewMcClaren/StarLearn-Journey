# StarLearn-Journey

Un app donde estudiantes o profesores pueden acceder a guias de aprendizaje personalizado para poder aprender cualquier tipo de tema educativo y pueden aprender habilidades de forma sencilla guiada con Inteligencia Artificial, para poder permitir que cualquier persona pueda aprender cualquier habilidad.

La idea es que siempre haya una nueva partida para que cada usuario pueda personalizar su personaje y pueda empezar a configurar si aprendizaje

Van a ver variables para el manejo de los jugadores:
Vidas
Puntos de Experiencia
Racha de Días
Mundo inicial
Monedas para comprar en tienda

Tienda: En esta tienda el usuario puede manejar sus objetos y demás accesorios
Mundos: Posibilidad de poder personalizar el fondo o tema de tu avatar

Será posible con los fragmentos de estrellas poder adquirir elementos de la tienda

Proximamente se definiran cómo se ve la aplicación en general, los colores principales, los botones y la barra de navegación.

Estarán presentes tarjetas de memoraización y juegos de preguntas para aumentar el nivel del juego.

Se integrará un modelo de inteligencia artificial para generar preguntar y procesar el reconocimiento de voz

Muchas veces lo que pasa es hacer que las trampas de juego permiten a los usuario crear nuevos mundos

Proximamente se habilitará posiblemente que el programa tenga tablas de usuarios, ranking de tiempo real de ligas y posibilidad de añadir amigos o funcionalidades de relacionamiento

Cada avatar tendrá una personalidad diferente y poder único que le permitirá poder avanzar más rápido en ciertos puntos del progreso.
